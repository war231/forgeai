## 1. init - 创建项目

### 语法

```bash
forgeai init [选项]
```

### 选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `--name` | 项目名称 | 必填 |
| `--genre` | 小说类型 | 必填 |
| `--mode` | 创作模式 | series |
| `--template` | 项目模板 | default |

### 示例

**基础示例：**

```bash
# 创建玄幻小说项目
forgeai init --name "我的玄幻小说" --genre "玄幻"
```

**进阶示例：**

```bash
# 创建都市小说项目（单本模式）
forgeai init --name "都市重生" --genre "都市" --mode single

# 使用模板创建项目
forgeai init --name "末世小说" --genre "科幻" --template apocalypse
```

## 2. write - 一键写作

### 语法

```bash

```

### 选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `--query` | 主题指导 | 无 |
| `--output` | 输出文件 | 自动生成 |
| `--smart` | 智能上下文 | false |

### 示例

**基础示例：**

```bash
# 创建第1章
forgeai write 1

# 带主题指导
forgeai write 1 --query "主角觉醒金手指"
```

**进阶示例：**

```bash
# 智能上下文模式
forgeai write 10 --query "感情线发展" --smart
```

## 3. check - 审查和处理

### 语法

```bash
forgeai check <子命令> <章节号> [选项]
```

### 子命令

| 子命令 | 说明 |
|--------|------|
| `after` | 写后处理 |
| `review` | 审查章节 |
| `before` | 写前检查 |

### 选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `--text-file` | 章节文件 | 必填 |
| `--independent` | 独立审查 | false |
| `--agents` | 指定Agent | 全部 |

### 示例

**基础示例：**

```bash
# 写后处理
forgeai check after 1 --text-file 4-正文/第1章.md

# 审查章节
forgeai check review 1
```

**进阶示例：**

```bash
# 独立审查模式
forgeai check review 1 --independent

# 只运行特定Agent
forgeai check review 1 --agents consistency,continuity

# 写前检查
forgeai check before 5
```

## 5. search - RAG检索

### 语法

```bash
forgeai search <查询词> [选项]
```

### 选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `--top-k` | 返回数量 | 5 |
| `--chapter` | 章节过滤 | 无 |

### 示例

**基础示例：**

```bash
# 搜索相关内容
forgeai search "主角修炼突破"
```

**进阶示例：**

```bash
# 限制返回数量
forgeai search "感情线" --top-k 10

# 指定章节范围
forgeai search "战斗场景" --chapter 1-10
```

## 命令组合使用

### 完整创作流程

```bash
# 1. 创建项目
forgeai init --name "我的小说" --genre "玄幻"
cd 我的小说

# 2. 写第1章
forgeai write 1 --query "主角觉醒"
# (使用AI生成内容，保存到 4-正文/第1章.md)

# 3. 写后处理
forgeai check after 1 --text-file 4-正文/第1章.md

# 4. 继续创作
forgeai write 2 --query "初次战斗"
forgeai check after 2 --text-file 4-正文/第2章.md

# 5. 查看状态
forgeai status
```

### 审查和修复流程

```bash
# 审查章节
forgeai check review 5 --independent

# 查看审查报告
cat 5-审查/第5章-审查报告.md

# (根据建议修改章节)

# 重新审查
forgeai check review 5
```

## 快捷命令

### 别名

ForgeAI 支持命令别名：

```bash
# init 别名
forgeai new --name "小说" --genre "玄幻"

# write 别名
forgeai w 1 --query "主题"

# check 别名
forgeai c after 1 --text-file 4-正文/第1章.md

# status 别名
forgeai s
```
