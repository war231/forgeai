# 写后处理

写后处理是ForgeAI工作流的最后阶段，包括全书一致性检查、统计报告生成、成稿导出和备份归档。

## 处理流程

```
1. 全书一致性检查
   ├─ 角色一致性
   ├─ 时间线连贯性
   └─ 情节逻辑一致性

2. 质量评估
   ├─ 追读力评估
   ├─ 节奏分析
   └─ 整体质量评分

3. 统计报告生成

4. 成稿导出

5. 备份归档
```

---

## 1. 全书一致性检查

```bash
# 检查所有一致性
forgeai review all --consistency

# 检查特定维度
forgeai review all --focus character    # 角色一致性
forgeai review all --focus timeline     # 时间线连贯性
forgeai review all --focus plot         # 情节逻辑

# 生成一致性报告
forgeai report consistency --output consistency-report.md
```

---

## 2. 质量评估

```bash
# 全书质量评估
forgeai review all --all-agents

# 生成质量报告
forgeai report quality --output quality-report.md

# 查看质量评分
forgeai show quality-score
```

**质量评分示例：**
```
整体质量评分: 8.2/10

分项评分:
  一致性: 9.0/10
  追读力: 7.8/10
  节奏感: 8.5/10
  高潮设置: 8.0/10
```

---

## 3. 统计报告

```bash
# 生成统计报告
forgeai report stats

# 详细统计
forgeai report stats --detailed

# 导出统计
forgeai report stats --output stats.json
```

**统计示例：**
```
项目统计
========

总章节数: 200章
总字数: 512,345字
平均章节字数: 2,562字

角色统计:
  主要角色: 3人
  配角: 15人

审查统计:
  发现问题: 156个
  已修复: 148个
  修复率: 94.9%
```

---

## 4. 成稿导出

```bash
# 导出为Markdown
forgeai export --format markdown --output 我的小说.md

# 导出为Word
forgeai export --format docx --output 我的小说.docx

# 导出为PDF
forgeai export --format pdf --output 我的小说.pdf

# 导出为EPUB
forgeai export --format epub --output 我的小说.epub

# 分卷导出
forgeai export --format markdown --split-by 50 --output 卷

# 批量导出多种格式
forgeai export --formats markdown,docx,pdf --output 我的小说
```

---

## 5. 备份归档

```bash
# 创建备份
forgeai backup create --name "最终版本"

# 查看备份列表
forgeai backup list

# 恢复备份
forgeai backup restore --name "最终版本"

# 导出完整项目
forgeai export-project --output 我的小说-项目备份.tar.gz

# 清理临时文件
forgeai cleanup
```
