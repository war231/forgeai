写前准备是ForgeAI创作流程中最关键的阶段。充分的准备能大幅提升创作效率和质量，减少后期修改工作量。

## 1. 世界观构建

### 1.1 时空背景

创建 `.forgeai/settings/世界观.yaml`：

```yaml
# 基本信息
name: 现代都市
time_period: 2024年
technology_level: 现代
genre: 都市职场

# 时空设定
time:
  start: 2024-01-01
  end: 2024-12-31
  timezone: "UTC+8 北京时间"
  calendar: 公历

space:
  main_location: 北京
  scope: 单城市为主，偶有出差
  travel_time:
    北京-上海: "5小时（高铁）"
    北京-深圳: "3小时（飞机）"
```

### 1.2 规则设定

```yaml
# 世界规则
world_rules:
  physical:
    - "遵循现实物理法则"
    - "无超自然元素"
    - "无科幻科技"

  social:
    - "遵循现实社会规则"
    - "法律制度与现实一致"
    - "文化背景为中国现代都市"

  technology:
    - "科技水平与现实一致"
    - "无未来科技"
    - "互联网、智能手机普及"

  constraints:
    - "角色不能有超能力"
    - "事件必须符合现实逻辑"
    - "时间线必须连贯"
```

### 1.3 地点定义

```yaml
# 地点设定
locations:
  # 主要地点
  北京:
    type: 城市
    role: 主要舞台
    climate: 温带季风
    population: 2100万
    districts:
      朝阳区:
        type: 商业区
        features:
          - CBD
          - 三里屯
          - 望京
      海淀区:
        type: 教育区
        features:
          - 中关村
          - 大学集中
          - 科技公司
      西城区:
        type: 历史区
        features:
          - 故宫
          - 天安门
          - 胡同

    landmarks:
      - name: 天安门
        type: 地标
        significance: 政治中心
      - name: 三里屯
        type: 商业区
        significance: 时尚购物

  # 次要地点
  上海:
    type: 城市
    role: 出差目的地
    distance_from_beijing: 1200公里

  # 具体场所
  公司:
    type: 建筑物
    location: 北京海淀区中关村
    floors: 20
    description: |
      一家中型互联网公司，位于中关村软件园。
      公司规模约500人，主营在线教育业务。
    areas:
      - 办公区
      - 会议室
      - 茶水间
      - 地下停车场

  李明家:
    type: 住宅
    location: 北京朝阳区望京
    type: 一居室租房
    description: |
      李明租住的一居室，位于望京某小区。
      距离公司约30分钟地铁车程。
```

## 2. 角色设计

### 2.1 主要角色

创建 `.forgeai/characters/李明.yaml`：

```yaml
# 基本信息
name: 李明
age: 25
gender: 男
occupation: 后端工程师
company: 星辰科技

# 外貌描写
appearance:
  height: 175cm
  weight: 65kg
  build: 偏瘦
  features:
    - 戴黑框眼镜
    - 黑色短发，略显凌乱
    - 目光锐利，眼神专注
    - 很少笑，表情严肃
  clothing_style:
    - 简单休闲
    - T恤牛仔裤
    - 偶尔穿格子衬衫

# 性格特征
personality:
  core_traits:
    - 谨慎：做事三思而后行
    - 聪明：逻辑思维强，学习能力强
    - 内向：不善社交，喜欢独处
    - 责任心强：对工作认真负责

  secondary_traits:
    - 固执：认准的事很难改变
    - 完美主义：追求代码和工作的完美
    - 敏感：对他人评价在意

  strengths:
    - 逻辑思维强：善于分析问题
    - 观察力敏锐：能注意到细节
    - 学习能力强：快速掌握新技术
    - 专注力强：能长时间专注工作

  weaknesses:
    - 社交能力弱：不善与人交流
    - 过度谨慎：有时错失机会
    - 不善表达情感：难以敞开心扉
    - 抗压能力一般：压力大了会焦虑

# 背景故事
background: |
  李明出生于普通工薪家庭，父亲是工程师，母亲是教师。
  从小对计算机感兴趣，大学计算机系毕业后，
  进入星辰科技担任后端工程师。

  性格内向，朋友不多，但工作能力出色。
  对代码质量要求很高，是团队的技术骨干。

  最近刚升职为技术组长，开始带团队，
  这对他来说是个不小的挑战。

# 能力值（1-10）
abilities:
  编程: 9
  逻辑思维: 8
  观察: 8
  学习: 8
  专注: 9
  社交: 3
  表达: 4
  运动: 4
  艺术: 2

# 当前状态
current_state:
  location: 北京
  mood: 平静
  goals:
    - 完成当前项目
    - 提升团队管理能力
    - 多交朋友

# 人际关系
relationships:
  王芳:
    type: 朋友
    closeness: 7
    description: 公司同事，为数不多的朋友
  张刚:
    type: 同事
    closeness: 5
    description: 同组工程师，关系一般
  李总:
    type: 上司
    closeness: 4
    description: 部门经理，工作关系
```

### 2.2 配角设定

创建 `.forgeai/characters/王芳.yaml`：

```yaml
name: 王芳
age: 24
gender: 女
occupation: 产品经理
company: 星辰科技

appearance:
  height: 165cm
  build: 匀称
  features:
    - 长发，经常扎马尾
    - 笑容灿烂
    - 眼睛明亮有神
    - 气质活泼开朗

personality:
  core_traits:
    - 开朗：性格外向，爱笑
    - 热情：对人友善，乐于助人
    - 细心：工作认真，注重细节
    - 善解人意：能理解他人感受

  strengths:
    - 沟通能力强：善于表达和倾听
    - 同理心强：能站在他人角度思考
    - 组织能力：善于协调资源

  weaknesses:
    - 有时过于乐观：低估困难
    - 情绪化：偶尔会情绪波动

background: |
  王芳是公司的产品经理，比李明晚半年入职。
  性格开朗，很快和同事们打成一片。

  注意到李明总是独来独往，主动和他交流，
  慢慢成为李明为数不多的朋友。

  对产品有独特见解，工作能力出色。

abilities:
  沟通: 9
  同理心: 8
  组织: 7
  编程: 3
  逻辑思维: 6

relationships:
  李明:
    type: 朋友
    closeness: 7
    description: 主动接近李明，成为朋友
```

### 2.3 关系网络

创建 `.forgeai/settings/关系网络.yaml`：

```yaml
# 角色关系网络
network:
  # 友谊关系
  friendships:
    - characters: [李明, 王芳]
      strength: 7
      since: chapter 10
      description: 王芳主动接近李明，成为朋友

  # 工作关系
  work_relationships:
    - characters: [李明, 张刚]
      type: 同事
      dynamic: 平等合作
    - characters: [李明, 李总]
      type: 上下级
      dynamic: 李明向李总汇报

  # 冲突关系
  conflicts:
    - characters: [李明, 神秘人]
      type: 对立
      since: chapter 30
      description: 李明发现神秘人的阴谋

# 关系发展计划
development:
  - chapter: 10
    event: "李明和王芳成为朋友"
    changes:
      - 李明 → 王芳: 朋友 (强度5→7)
      - 王芳 → 李明: 朋友 (强度5→7)

  - chapter: 50
    event: "李明和王芳关系升温"
    changes:
      - 李明 → 王芳: 好友 (强度7→9)
```

## 3. 情节规划

### 3.1 主线大纲

创建 `.forgeai/plots/main-plot.yaml`：

```yaml
name: 主线情节
genre: 都市职场
theme: 成长与蜕变

# 三幕结构
structure:
  # 第一幕：铺垫（1-50章）
  first_act:
    chapters: 1-50
    theme: "新人入职，适应环境"
    percentage: 25%

    setup:
      - chapter: 1
        event: "李明入职星辰科技"
        significance: "激励事件"
      - chapter: 5
        event: "展现工作能力，获得认可"
      - chapter: 10
        event: "遇到王芳，建立友谊"
      - chapter: 20
        event: "升职为技术组长"
      - chapter: 30
        event: "发现公司异常数据"
        significance: "转折点"
      - chapter: 40
        event: "暗中调查，收集线索"
      - chapter: 50
        event: "决定深入调查真相"
        significance: "第一幕高潮"

  # 第二幕：对抗（51-150章）
  second_act:
    chapters: 51-150
    theme: "调查真相，成长蜕变"
    percentage: 50%

    confrontation:
      - chapter: 60
        event: "获得关键证据"
      - chapter: 80
        event: "遭遇警告和威胁"
      - chapter: 100
        event: "陷入困境，几乎放弃"
        significance: "中点低谷"
      - chapter: 120
        event: "王芳鼓励，重拾信心"
      - chapter: 140
        event: "找到决定性证据"
      - chapter: 150
        event: "真相大白"
        significance: "第二幕高潮"

  # 第三幕：解决（151-200章）
  third_act:
    chapters: 151-200
    theme: "解决问题，新的开始"
    percentage: 25%

    resolution:
      - chapter: 160
        event: "公开真相，对抗反派"
      - chapter: 170
        event: "公司整改，正义伸张"
      - chapter: 180
        event: "李明成长，性格转变"
      - chapter: 190
        event: "与王芳关系确定"
      - chapter: 200
        event: "新的开始"
        significance: "结局"
```

## 4. 素材准备

### 4.1 样章参考

```bash
# 导入样章
forgeai import sample-chapter.md --as-template

# 或创建模板
forgeai create template --genre 都市 --style 轻松
```

### 4.2 知识库

创建 `.forgeai/knowledge/` 目录，添加参考素材：

```
knowledge/
├── 职场知识.md        # 职场相关知识
├── 技术知识.md        # 编程技术知识
├── 北京地理.md        # 北京地理信息
└── 生活常识.md        # 生活常识
```

### 4.3 风格指南

在 `.forgeai/config.yaml` 中设置：

```yaml
writing:
  # 写作风格
  style: 轻松幽默
  tone: 积极向上

  # 叙事视角
  perspective: 第三人称
  focus: 主角视角为主

  # 对话风格
  dialogue:
    style: 自然对话
    避免过度书面化
    符合角色性格

  # 描写风格
  description:
    动作描写: 简洁有力
    心理描写: 细腻深入
    环境描写: 适度即可
```

## 准备检查清单

完成写前准备后，使用此清单检查：

### ✅ 世界观
- [ ] 时空背景已定义
- [ ] 世界规则已设定
- [ ] 主要地点已创建

- [ ] 人际关系已定义

### ✅ 情节
- [ ] 主线大纲已规划
- [ ] 三幕结构已设计
- [ ] 关键节点已标注

### ✅ 素材
- [ ] 写作风格已设定
- [ ] 知识库已准备（可选）
