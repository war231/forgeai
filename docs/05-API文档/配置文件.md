# 配置文件详解

ForgeAI使用YAML格式的配置文件，位于项目根目录的 `.forgeai/config.yaml`。

## 配置文件结构

```yaml
# API配置
api:
  provider: openai
  key: "your-api-key"
  model: "gpt-4"
  base_url: "https://api.openai.com/v1"

# 项目配置
project:
  name: "我的小说"
  genre: "都市"
  target_words: 500000
  target_chapters: 200

# 写作配置
writing:
  words_per_chapter: 2500
  style: "轻松幽默"
  perspective: "第三人称"
  tone: "积极向上"

# 审查配置
review:
  auto_review: true
  agents:
    - consistency-checker
    - ooc-checker
    - continuity-checker
    - reader-pull-checker
  severity_threshold: "moderate"

# Agent配置
agents:
  ooc-checker:
    sensitivity: medium
    capability_tolerance: 10%
  continuity-checker:
    time_jump_tolerance: 1
    check_spatial_jump: true
  reader-pull-checker:
    hook_threshold: 5
    suspense_density: 3

# 数据配置
data:
  data_path: ".forgeai"
  auto_backup: true
  backup_interval: 1
  backup_retention: 10

# 日志配置
logging:
  level: INFO
  file: ".forgeai/logs/forgeai.log"
  format: "%(asctime)s - %(name)s - %(levelname)s - %(message)s"
```

## API配置 (api)

### api.provider

API提供商。

```yaml
api:
  provider: openai  # openai, anthropic, custom
```

**可选值：**
- `openai`: OpenAI API
- `anthropic`: Anthropic API
- `custom`: 自定义API

### api.key

API密钥。

```yaml
api:
  key: "your-api-key"
```

**注意：** 也可以通过环境变量 `FORGEAI_API_KEY` 设置。

### api.model

使用的模型。

```yaml
api:
  model: "gpt-4"  # gpt-4, gpt-3.5-turbo, claude-3, etc.
```

**推荐模型：**
- `gpt-4`: 质量最高，推荐用于创作
- `gpt-3.5-turbo`: 速度快，适合测试
- `claude-3`: Anthropic的模型

### api.base_url

API基础URL。

```yaml
api:
  base_url: "https://api.openai.com/v1"
```

**注意：** 使用代理或自定义API时需要修改此配置。

## 项目配置 (project)

### project.name

项目名称。

```yaml
project:
  name: "我的小说"
```

### project.genre

小说类型。

```yaml
project:
  genre: "都市"  # 都市, 玄幻, 武侠, 科幻, 悬疑, etc.
```

### project.target_words

目标字数。

```yaml
project:
  target_words: 500000  # 50万字
```

### project.target_chapters

目标章节数。

```yaml
project:
  target_chapters: 200  # 200章
```

### project.author

作者信息。

```yaml
project:
  author:
    name: "作者名"
    email: "author@example.com"
```

## 写作配置 (writing)

### writing.words_per_chapter

每章字数。

```yaml
writing:
  words_per_chapter: 2500  # 每章2500字
```

### writing.style

写作风格。

```yaml
writing:
  style: "轻松幽默"  # 轻松幽默, 严肃深沉, 紧张悬疑, etc.
```

**常见风格：**
- `轻松幽默`: 轻松愉快的风格
- `严肃深沉`: 严肃深刻的风格
- `紧张悬疑`: 紧张刺激的风格
- `温馨治愈`: 温暖治愈的风格
- `热血激昂`: 热血激昂的风格

### writing.perspective

叙事视角。

```yaml
writing:
  perspective: "第三人称"  # 第一人称, 第三人称
```

### writing.tone

整体基调。

```yaml
writing:
  tone: "积极向上"  # 积极向上, 悲观消极, 中性客观
```

### writing.dialogue_style

对话风格。

```yaml
writing:
  dialogue_style:
    style: "自然对话"
    avoid_overly_written: true
    match_character: true
```

### writing.description_ratio

描写比例。

```yaml
writing:
  description_ratio:
    action: 0.3      # 动作描写 30%
    dialogue: 0.3    # 对话描写 30%
    psychology: 0.2  # 心理描写 20%
    environment: 0.1 # 环境描写 10%
    exposition: 0.1  # 说明描写 10%
```

## 审查配置 (review)

### review.auto_review

是否自动审查。

```yaml
review:
  auto_review: true  # 创作后自动审查
```

### review.agents

启用的审查Agent。

```yaml
review:
  agents:
    - consistency-checker   # 设定一致性检查
    - ooc-checker          # OOC检测
    - continuity-checker   # 连贯性检查
    - reader-pull-checker  # 追读力检查
    - high-point-checker   # 高潮点检查
    - pacing-checker       # 节奏检查
```

### review.severity_threshold

问题严重度阈值。

```yaml
review:
  severity_threshold: "moderate"  # minor, moderate, serious, critical
```

**说明：** 只显示和修复此级别及以上问题。

### review.auto_fix

是否自动修复问题。

```yaml
review:
  auto_fix: true  # 自动修复简单问题
```

## Agent配置 (agents)

### agents.ooc-checker

OOC检测Agent配置。

```yaml
agents:
  ooc-checker:
    sensitivity: medium              # 检测敏感度
    capability_tolerance: 10%        # 能力容忍度
    check_speech_pattern: true       # 检测说话方式
    allow_character_growth: true     # 允许角色成长
    growth_threshold: 10             # 成长判定阈值
    ignore_types:                    # 忽略的OOC类型
      - "梦境"
      - "幻觉"
```

### agents.continuity-checker

连贯性检查Agent配置。

```yaml
agents:
  continuity-checker:
    time_jump_tolerance: 1           # 时间跳跃容忍度（章节数）
    check_spatial_jump: true         # 检测空间跳跃
    causal_chain_depth: 3            # 因果链检测深度
    plot_jump_threshold: medium      # 情节跳跃检测阈值
    check_emotion_continuity: true   # 检测情绪连贯性
    emotion_tolerance: 2             # 情绪变化容忍度
    ignore_scenes:                   # 忽略的场景类型
      - "梦境"
      - "回忆"
```

### agents.reader-pull-checker

追读力检查Agent配置。

```yaml
agents:
  reader-pull-checker:
    hook_threshold: 5                # 钩子强度阈值
    suspense_density: 3              # 悬念密度阈值
    conflict_tolerance: 2            # 冲突缺失容忍度
    info_release_span: 3             # 信息释放建议
    abandonment_model: conservative  # 弃书风险预测模型
    check_emotional_hooks: true      # 情感钩子检测
    check_expectation_management: true  # 预期管理检测
```

### agents.high-point-checker

高潮点检查Agent配置。

```yaml
agents:
  high-point-checker:
    intensity_threshold: 7           # 高潮强度阈值
    recommended_interval: 3-5        # 高潮间隔建议
    consecutive_tolerance: 2         # 连续高潮容忍度
    buildup_chapters: 3-5            # 铺垫章节数建议
    aftermath_chapters: 1-2          # 余波处理建议
    detect_types: true               # 高潮类型检测
    structure_pattern: "three_act"   # 结构模式
    generate_chart: true             # 生成高潮曲线图
```

### agents.pacing-checker

节奏检查Agent配置。

```yaml
agents:
  pacing-checker:
    pacing_standards:
      fast:
        max_words_per_event: 500
        min_events_per_chapter: 3
      medium:
        max_words_per_event: 1000
        min_events_per_chapter: 2
      slow:
        max_words_per_event: 2000
        min_events_per_chapter: 1
    scene_length:
      battle: [1500, 3000]
      dialogue: [1000, 2000]
      daily: [800, 1500]
    paragraph_pacing: true           # 段落节奏检测
    chapter_pacing: true             # 章节节奏检测
    generate_chart: true             # 生成节奏曲线图
```

## 数据配置 (data)

### data.data_path

数据存储路径。

```yaml
data:
  data_path: ".forgeai"  # 默认在项目根目录
```

### data.auto_backup

是否自动备份。

```yaml
data:
  auto_backup: true
```

### data.backup_interval

备份间隔（天）。

```yaml
data:
  backup_interval: 1  # 每天备份一次
```

### data.backup_retention

备份保留数量。

```yaml
data:
  backup_retention: 10  # 保留最近10个备份
```

### data.compress_backup

是否压缩备份。

```yaml
data:
  compress_backup: true
```

## 环境变量覆盖

配置文件中的值可以被环境变量覆盖：

```bash
# API密钥
export FORGEAI_API_KEY="your-api-key"

# API模型
export FORGEAI_MODEL="gpt-4"

# API基础URL
export FORGEAI_BASE_URL="https://api.openai.com/v1"

# 配置文件路径
export FORGEAI_CONFIG="path/to/config.yaml"

# 日志级别
export FORGEAI_LOG_LEVEL="INFO"
```

```yaml

api:
  provider: openai
  key: "your-api-key"

  base_url: "https://api.openai.com/v1"
```
