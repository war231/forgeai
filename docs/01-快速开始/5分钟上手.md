# 5分钟上手 ForgeAI

> **从零开始创作你的第一部AI辅助小说**

---

## 前置要求

- Python 3.8+
- OpenAI/Kimi/DeepSeek API Key

---

## 第一步：安装 (1分钟)

### 方式一：从源码安装（推荐）

```bash
# 克隆仓库
git clone https://github.com/yourusername/forge-ai.git
cd forge-ai

# 安装依赖
pip install -r requirements.txt

# 安装 ForgeAI
pip install -e .
```

### 方式二：直接安装

```bash
pip install forgeai
```

### 验证安装

```bash
forgeai --version
# 输出: ForgeAI v1.0.0
```

---

## 第二步：创建项目 (1分钟)

```bash
# 创建新项目
forgeai init --name "我的玄幻小说" --genre "玄幻"

# 进入项目目录
cd 我的玄幻小说
```

**创建的目录结构：**

```
我的玄幻小说/
├── 1-边界/          # 边界条件（题材、风格、目标读者）
├── 2-设定/          # 世界观、角色、金手指设定
├── 3-大纲/          # 章节规划、情节节奏
├── 4-正文/          # 章节正文
├── 5-审查/          # 审查报告
├── config.json      # 配置文件
└── SOLOENT.md       # 控制面板
```

---

## 第三步：配置 API (1分钟)

编辑 `config.json`：

```json
{
  "project": {
    "name": "我的玄幻小说",
    "genre": "玄幻",
    "mode": "series"
  },
  "llm": {
    "provider": "openai",
    "model": "gpt-4o-mini",
    "api_key": "your-api-key-here",
    "base_url": "https://api.openai.com/v1"
  },
  "rag": {
    "enabled": true,
    "chunk_size": 500,
    "chunk_overlap": 50
  }
}
```

**支持的LLM提供商：**

- OpenAI (gpt-4o, gpt-4o-mini)
- Kimi (moonshot-v1-8k, moonshot-v1-32k)
- DeepSeek (deepseek-chat)

## 第五步：写后处理 (自动)

```bash
# 写后自动处理（索引、实体提取、评分、审查）
forgeai check after 1 --text-file 4-正文/第1章.md
```

**自动执行：**

1. ✅ 索引章节（RAG）
2. ✅ 提取实体（人物/地点/物品）
3. ✅ AI味评分（24种特征检测）
4. ✅ 更新进度
5. ✅ 9个Agent并行审查

## 常见问题

### Q: 如何选择LLM提供商？

**A:** 根据你的需求选择：

- **OpenAI GPT-4o**: 质量最高，成本较高
- **Kimi**: 中文效果好，成本适中
- **DeepSeek**: 成本最低，质量稳定

### Q: 如何提高AI味评分？

**A:** AI味评分检测24种AI写作特征，评分越高越像人类写作。可以通过以下方式提高：

1. 避免使用"令人...的是"、"不言而喻"等AI常用表达
2. 减少三段式结构
3. 增加对话和动作描写
4. 使用更自然的中文表达

### Q: Agent审查不通过怎么办？

**A:** 每个Agent都会提供具体的问题说明和修复建议：

1. 查看 `5-审查/第N章-审查报告.md`
2. 根据建议修改章节内容
3. 重新运行 `forgeai check after N`
