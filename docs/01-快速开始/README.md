## 什么是 ForgeAI？

ForgeAI 是一个基于多Agent协作的长篇小说AI创作系统，通过以下技术实现高质量AI辅助创作：

- **多Agent协作审查** - 9个专业Agent并行审查，确保质量
- **RAG语义检索** - 智能召回伏笔、角色状态、相关章节
- **自动化流水线** - 写前检查、上下文提取、写后处理全自动
- **样板书学习** - 分析对标作品，提取套路/节奏/文风
- **去AI痕迹** - 24种AI写作特征检测，保持文风自然

## 核心功能

### 1. 一键写作

```bash
# 创建项目
forgeai init --name "我的小说" --genre "玄幻"

# 开始写作（自动完成写前检查、上下文提取、提示词生成）
forgeai write 1 --query "主角觉醒金手指"
```

### 2. 智能审查

```bash
# 写后自动审查（9个Agent并行审查）
forgeai check after 1 --text-file 4-正文/第1章.md

# 独立审查模式
forgeai check review 1 --independent
```

### 3. 实体管理

```bash
# 查看实体
forgeai entity list --type character

# 添加实体
forgeai entity add "李天" --type character --tier core
```

### 4. 伏笔管理

```bash
# 查看活跃伏笔
forgeai foreshadow list --active

# 添加伏笔
forgeai foreshadow add "神秘玉佩" --chapter 1 --payoff 20
```
