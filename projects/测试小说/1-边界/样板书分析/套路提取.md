# 套路提取

## 样板书信息
- 书名：测试样板书
- 分析章节：第1-10章

## 1. 主线套路

**套路类型**：升级流
- 核心矛盾：主角实力不足，被压制
- 解决模式：获得金手指 → 快速升级 → 打脸反派
- 循环周期：每5章一个小循环

## 2. 开篇套路（黄金三章）

**第1章**：
- 钩子类型：危机开场
- 金手指亮相方式：意外获得

**第2章**：
- 首胜类型：小试牛刀
- 期待感建立：展示金手指潜力

**第3章**：
- 世界观展开：力量体系介绍
- 主线确立：复仇/崛起目标

## 3. 付费点设计

**付费点位置**：第15章
- 前置钩子：主角即将突破关键境界
- 截断位置：战斗高潮前
- 期待感：读者想看主角如何突破

## 4. 周期性套路

**小周期（2-3章）**：
- 遇到小反派 → 被嘲讽 → 悄悄提升 → 打脸

**中周期（8-10章）**：
- 遇到中BOSS → 被压制 → 获得机缘 → 反杀

**大周期（20-30章）**：
- 换地图 → 新环境 → 新挑战 → 新高度
