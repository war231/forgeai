# 新书设定案

## 1. 世界观

### 1.1 时代背景

- **朝代**：东汉建安年间（196-220年）
- **政治格局**：曹操挟天子以令诸侯，袁绍据河北，孙策据江东，刘备依附刘表
- **核心冲突**：诸侯争霸 + 主角身份危机

### 1.2 力量体系

- **无玄幻元素**，纯历史权谋
- **力量来源**：智谋、人脉、兵权、粮草

### 1.3 地理格局

```
北方：袁绍（河北四州）
中原：曹操（兖州、豫州、司隶）
南方：孙策（江东）、刘表（荆州）、刘备（依附刘表）
西方：马腾（凉州）、张鲁（汉中）、刘璋（益州）
```

## 3. 金手指

### 3.1 核心机制

| 维度 | 内容 |
|-----|------|
| **类型** | 历史先知 + 现代战略思维 |
| **表现** | 知道历史走向（谁会赢、谁会死、关键战役结果） |

### 3.2 代价与限制

| 限制 | 说明 |
|-----|------|
| **蝴蝶效应** | 改变太多会导致后续不可预测 |
| **记忆偏差** | 正史vs演义vs真实历史有出入 |
| **伪装需求** | 必须装作"推演得出"，不能暴露穿越者身份 |

### 3.3 进化路线

```
1.0 被动预知 → 2.0 主动推演 → 3.0 改变历史（金手指逐渐失效）
```

## 4. 主线推演

### 4.1 分卷规划

| 卷 | 章节 | 核心剧情 | 爽点类型 |
|----|------|---------|---------|
| **第一卷：潜龙勿用** | 1-30章 | 投奔曹操 → 被郭嘉脑补 → 预言吕布必败 | 迪化误解 + 扮猪吃虎 |
| **第二卷：见龙在田** | 31-80章 | 官渡之战 → 预言袁绍必败 → 身份危机 | 打脸权威 + 身份掉马危机 |
| **第三卷：飞龙在天** | 81-150章 | 赤壁之战 → 劝曹操勿攻 → 救曹操脱险 | 越级反杀 |
| **第四卷：亢龙有悔** | 151-200章 | 曹操称魏公 → 主角劝阻 → 最终选择 | 权谋博弈 |

### 4.2 关键事件

| 事件 | 章节 | 主角行动 | 结果 |
|-----|------|---------|------|
| **吕布覆灭** | 10-15章 | 预言吕布必败 | 曹操信任+10 |
| **官渡之战** | 40-60章 | 预言袁绍必败，建议烧乌巢 | 曹操信任+20，郭嘉脑补+30 |
| **赤壁之战** | 100-130章 | 劝曹操勿攻，失败，救曹操脱险 | 曹操信任+50，但被冷落 |
| **郭嘉之死** | 80章 | 试图救郭嘉 | 成功/失败（待定） |

## 5. 关键关系博弈

| 角色 | 与主角关系 | 博弈逻辑 |
|-----|----------|---------|
| **曹操** | 主公 | 曹操：此人可用但需防备 → 主角：要被信任但不能太露 |
| **郭嘉** | 脑补者 | 郭嘉：此人深不可测 → 主角：我只是随口一说... |
| **荀彧** | 审视者 | 荀彧：颍川陈氏无此人 → 主角：我是旁支，族谱可查 |
| **程昱** | 竞争者 | 程昱：此人装神弄鬼 → 主角：用事实打脸 |
