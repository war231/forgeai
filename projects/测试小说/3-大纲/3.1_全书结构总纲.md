## 二、节奏公式

### 2.1 黄金三章公式

```
第1章：危机开场 + 身份切换钩子（0-1爽点）
├─ 开场：主角穿越，被当作流民
├─ 危机：如何活下去？
└─ 钩子：冒充颍川陈氏，能否骗过？

第2章：迪化误解 + 扮猪吃虎（3-4爽点）
├─ 见曹操：被审视
├─ 郭嘉脑补：此人深不可测！
└─ 主角内心：我只是随口一说...

第3章：事实打脸 + 新钩子（3-4爽点）
├─ 预言验证：吕布必败
├─ 曹操信任+10
└─ 新钩子：荀彧怀疑，身份危机
```

### 2.2 周期性节奏

```
【小周期】3-5章（铺垫→高潮）
示例：第1章铺垫 → 第2-3章高潮 → 第4章过渡

【中周期】10-15章（事件完整）
示例：第6-15章（吕布覆灭事件）

【大周期】30-50章（一卷）
示例：第一卷（潜龙勿用）
```

### 2.3 爽点密度控制

| 章节类型 | 爽点数 | 功能 | 示例 |
|---------|--------|------|------|
| **铺垫章** | 0-2个 | 建立期待，埋伏笔 | 第1章、第5章 |
| **过渡章** | 2-3个 | 推进剧情，转换场景 | 第6章、第16章 |
| **高潮章** | 4-6个 | 爆发爽感，关键转折 | 第2章、第15章 |

## 四、启示

### 4.1 核心卖点

1. **迪化流核心**：爽感来自"别人脑补我是高人"
2. **历史先知**：主角知道三国历史走向
3. **身份危机**：必须伪装成"颍川陈氏旁支"
4. **改变历史**：能否救郭嘉？能否阻止赤壁之败？

### 4.2 创作原则

| 原则 | 说明 |
|-----|------|
| **禁止降智** | 曹操/郭嘉/荀彧等智谋人物不得降智 |
| **禁止OOC** | 历史人物性格必须符合史书记载 |
| **禁止爽文套路** | 不得出现"主角一句话改变战局"等不合理情节 |
| **禁止历史虚无** | 大事件走向必须符合史实（除非主角主动改变） |
| **迪化流核心** | 爽感必须来自"被脑补"，而非主角主动装逼 |

### 4.3 关键分支点

| 分支点 | 章节 | 选择 | 影响 |
|--------|------|------|------|
| **救郭嘉？** | 第80章 | 成功/失败 | 影响后续剧情走向 |
| **赤壁劝谏** | 第100章 | 成功/失败 | 影响曹操对主角的信任 |
| **最终选择** | 第180章 | 归隐/决裂/改变 | 决定结局走向 |
