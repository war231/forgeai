> **来源**：Webnovel Writer reading-power-taxonomy.md

## 一、钩子类型 (Hook Types)

### 1.1 五类钩子

| 类型 | 驱动力 | 适用场景 |
|------|--------|---------|
| **危机钩** | 危险逼近，读者担心 | 爽文/悬疑 |
| **悬念钩** | 信息缺口，读者好奇 | 悬疑/日常 |
| **情绪钩** | 强情绪触发（愤怒/心疼/心动）| 言情/爽文 |
| **选择钩** | 两难抉择，想知道选择 | 悬疑/言情 |
| **渴望钩** | 好事将至，读者期待 | 爽文/言情 |

### 1.2 渴望钩子类型

| 子类型 | 示例 |
|--------|------|
| 成长渴望 | "再有三天，就能突破了" |
| 关系渴望 | "她答应了那个约定" |
| 复仇渴望 | "王少不知道，他的末日近在眼前" |
| 真相渴望 | "答案或许就在这块玉佩里" |
| 收获渴望 | "这朵火莲，足以让他脱胎换骨" |

### 1.3 钩子强度

| 强度 | 适用场景 |
|------|---------|
| **strong** | 卷末/关键转折/大冲突前 |
| **medium** | 普通剧情章 |
| **weak** | 过渡章/铺垫章 |

### 1.4 章内 vs 章末

| 位置 | 常用钩子 | 作用 |
|------|---------|------|
| **章内** | 悬念钩、情绪钩 | 保持沉浸 |
| **章末** | 危机钩、选择钩、渴望钩 | 驱动点下一章 |

## 二、爽点模式 (Cool-Point Patterns)

### 2.1 八种标准模式

| 模式 | 特征 |
|------|------|
| 装逼打脸 | 嘲讽→反转→震惊 |
| 扮猪吃虎 | 示弱→暴露→碾压 |
| 越级反杀 | 差距→策略→逆转 |
| 打脸权威 | 权威→挑战→成功 |
| 反派翻车 | 得意→反杀→落幕 |
| 甜蜜超预期 | 期待→超预期→升华 |
| 迪化误解 | 随意行为+信息差+脑补+读者优越 |
| 身份掉马 | 长期隐藏+触发事件+揭露+群体反应 |

### 2.2 结构提示（30/40/30）

| 阶段 | 占比 | 作用 |
|------|------|------|
| 铺垫 | ~30% | 建立信息差、压力、期待 |
| 兑现 | ~40% | 核心爽点执行 |
| 余波 | ~30% | 反应、收获、新期待 |

## 三、微兑现 (Micro-Payoff)

### 3.1 微兑现类型

| 类型 | 示例 |
|------|------|
| 信息兑现 | "原来那把钥匙的真正用途是..." |
| 关系兑现 | "她第一次主动握住了他的手" |
| 能力兑现 | "他终于掌握了这门功法的精髓" |
| 资源兑现 | "储物袋里竟然还藏着一颗聚气丹" |
| 认可兑现 | "在场所有人看他的眼神都变了" |
| 情绪兑现 | "他终于说出了压在心底的那句话" |
| 线索兑现 | "三年前的那件事，终于有了眉目" |

### 3.2 题材偏好

| 题材 | 偏好微兑现 | 每章建议 |
|------|-----------|---------|
| 爽文 | 能力/资源/认可 | 2-3 |
| 言情 | 关系/情绪/认可 | 1-2 |
| 悬疑 | 信息/线索 | 1-2 |
| 日常 | 关系/情绪 | 1 |

## 四、约束分层

### 硬约束

| ID | 约束 | 触发条件 |
|----|------|---------|
| HARD-001 | 可读性底线 | 看不懂"发生了什么" |
| HARD-002 | 承诺违背 | 上章钩子完全无回应 |
| HARD-003 | 节奏灾难 | 连续N章无推进 |
| HARD-004 | 冲突真空 | 整章无问题/目标/代价 |
