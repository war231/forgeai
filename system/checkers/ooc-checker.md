# 人物OOC检查器 (OOC Checker)

> **职责**：角色完整性守卫者，防止 OOC（Out-Of-Character）违规。

## 执行流程

### Step 1: 加载角色档案

从 `2-设定/2.4_主要角色设定表.md` 读取：
- 性格特征
- 说话风格
- 核心价值观
- 行为倾向

### Step 2: 行为采样

对每章提取角色的动作和对话

### Step 3: 三级OOC判定

#### 一级：轻微偏离（可接受）
角色行为略有不同，但有合理的世界观内解释。

```
✓ 平时冷静的角色，家人被威胁时暴怒 → 触及底线，合理
✓ 温柔的角色，生死关头展现强势 → 危机激发隐藏面，有铺垫
```

#### 二级：中度失真（需警告）
行为不一致，缺乏充分铺垫或解释。

```
⚠️ 三思而行的角色，普通挑衅时突然冲动 → 缺少动机
⚠️ 高傲冷漠的角色，对路人突然温柔 → 性格转变过快
```

#### 三级：严重崩坏（必须修复）
行为与既定特征完全相反，且无任何解释。

```
❌ 反派（智商在线）突然犯低级错误 → 为剧情服务
❌ 隐忍冷静的角色，无刺激下持续多章冲动 → 核心人设崩塌
```

### Step 4: 对话风格检查

| 角色类型 | 预期风格 | OOC示例 |
|---------|---------|---------|
| 冷静型主角 | 言简意赅、语气平淡 | ❌ "哈哈哈！老子今天就让你见识！" |
| 嚣张型反派 | 嘲讽、轻蔑、自信 | ❌ "对不起...我错了..." |
| 修仙者 | "阁下/道友/在下" | ❌ "牛逼/666/OMG" |

### Step 5: 角色成长 vs OOC

```
✓ 成长：第1-10章谨慎（实力弱）→ 第50章自信果敢（实力提升+磨练）→ 合理
❌ OOC：第10章隐忍 → 第11章话痨 → 无解释的性格突变
```

成长检查清单：
- [ ] 性格转变有合理触发事件？
- [ ] 转变过程有渐进式铺垫？
- [ ] 转变后的行为与触发事件逻辑一致？

### Step 6: 生成报告

```markdown
# 人物OOC检查报告

## 主要角色行为采样
### {角色名}
| 章节 | 行为/对话 | 人设匹配 | OOC级别 |

## 对话风格检查
| 角色 | 预期风格 | 发现违规 |

## 性格转变检查
| 角色 | 原有特征 | 当前特征 | 合理性 | 判定 |

## 修复建议
1. **修复OOC**: ...
2. **补充铺垫**: ...

## 综合评分
- 严重OOC: {count}
- 中度OOC: {count}
- 轻微OOC: {count}
**结论**: {通过/警告/未通过}
```

## 禁止事项

❌ 通过存在严重OOC且未标记的章节
❌ 忽略角色对话风格违规
❌ 混淆OOC与角色成长
