# 言情小说情绪设计 (Romance Emotion Design)

> **核心原则**: 言情的核心是情绪。读者要的是"心动、心疼、心酸、心甜"的情绪体验。

---

## 一、言情四要素

### 1.1 四要素定义

| 要素 | 定义 | 读者体验 |
|------|------|----------|
| **心动** | 初遇、好感萌芽 | 甜蜜、期待 |
| **心疼** | 误会、分离、牺牲 | 酸涩、共情 |
| **心酸** | 阻碍、错过、遗憾 | 惋惜、纠结 |
| **心甜** | 互动、告白、在一起 | 满足、幸福 |

### 1.2 情绪比例

| 阶段 | 心动 | 心疼 | 心酸 | 心甜 |
|------|------|------|------|------|
| 相遇期 | 60% | 10% | 20% | 10% |
| 接近期 | 40% | 20% | 20% | 20% |
| 冲突期 | 20% | 40% | 30% | 10% |
| 解决期 | 20% | 10% | 10% | 60% |

## 二、言情节拍设计

### 2.1 互动节拍

**定义**: 男女主之间的互动单元

**类型**:

| 类型 | 定义 | 效果 |
|------|------|------|
| **意外接触** | 无意的身体接触 | 心动 |
| **保护行为** | 一方保护另一方 | 心动+心甜 |
| **吃醋反应** | 因对方与异性互动而吃醋 | 心甜 |
| **关心细节** | 发现对方的小习惯/需求 | 心甜 |
| **误会冲突** | 因误会产生的冲突 | 心疼+心酸 |
| **真相揭露** | 误会解开 | 心甜释放 |
| **告白场景** | 表白心意 | 心甜高潮 |

### 2.2 期待-满足循环

```
建立期待 → 延迟满足 → 强化期待 → 满足 → 新期待
   ↓           ↓           ↓          ↓        ↓
 "想见面"   "见不到"    "终于能见"  "见面了"  "想更进一步"
```

## 三、言情爽点类型

### 3.1 甜蜜爽点

| 类型 | 公式 | 示例 |
|------|------|------|
| **双向奔赴** | 双方同时主动 | 两人同时走向对方 |
| **偏爱展示** | 只对她/他不一样 | 高冷男主只为女主破例 |
| **守护时刻** | 关键时刻守护 | 危险来临时本能保护 |
| **吃醋表现** | 因对方而失控 | 一向冷静的他突然失控 |
| **宣示主权** | 对外宣示关系 | "她是我的人" |

### 3.2 虐心设计

| 类型 | 公式 | 适用阶段 |
|------|------|----------|
| **误会虐** | 善意被误解 | 中期 |
| **选择虐** | 爱与现实冲突 | 中后期 |
| **分离虐** | 被迫分开 | 后期 |
| **错过虐** | 差一点就在一起 | 高潮前 |
| **牺牲虐** | 为对方牺牲 | 高潮 |

### 3.3 反转爽点

| 类型 | 定义 | 效果 |
|------|------|------|
| **误会反转** | 误会解开，原来对方一直爱我 | 大爽点 |
| **身份反转** | 那个讨厌的人其实一直在帮我 | 大爽点 |
| **态度反转** | 高冷变温柔，冷漠变深情 | 中爽点 |

## 四、言情人物设计

### 4.1 男主类型

| 类型 | 特点 | 爽点来源 |
|------|------|----------|
| **高冷型** | 外冷内热 | 反差萌、专属温柔 |
| **腹黑型** | 算计一切唯独对她真心 | 被宠爱感 |
| **霸道型** | 强势但尊重她 | 安全感 |
| **温柔型** | 全方位的好 | 被呵护感 |
| **傲娇型** | 嘴硬心软 | 期待他承认 |

### 4.2 女主类型

| 类型 | 特点 | 爽点来源 |
|------|------|----------|
| **软萌型** | 需要保护 | 激发保护欲 |
| **独立型** | 自强不息 | 成长爽感 |
| **聪明型** | 智商在线 | 智斗爽感 |
| **傲娇型** | 嘴硬心软 | 等她承认 |

### 4.3 CP化学反应

**公式**: 性格差异 + 互补需求 + 共同经历 = CP感

```
高冷男主 + 软萌女主 = 护短宠溺
腹黑男主 + 聪明女主 = 智者对决
霸道男主 + 独立女主 = 从对抗到征服
```

## 五、言情场景模板

### 5.1 心动场景

```
场景：意外接触
要素：无意 → 意识到 → 心跳加速 → 尴尬回避

示例：
她脚下一滑，整个人向前栽去。
预想中的疼痛没有到来——一只有力的手揽住了她的腰。
她抬起头，正对上他深邃的眼眸。
时间仿佛静止。
"小心。"他的声音有些低沉。
她猛地站直，脸红得像熟透的苹果："谢、谢谢。"
```

### 5.2 告白场景

```
场景：告白
要素：铺垫 → 决定 → 告白 → 反应 → 结果

示例：
他看着她，眼神前所未有的认真。
"我有话想对你说。"
"什么？"
"我喜欢你。不是朋友那种喜欢。"
她愣住了，心跳快得像要跳出胸腔。
"你...你说什么？"
他走近一步，认真地看着她的眼睛："我说，我喜欢你。想让你做我女朋友的那种喜欢。"
```

### 5.3 吃醋场景

```
场景：吃醋
要素：触发 → 反应 → 质问 → 真相/和好

示例：
"那个男生是谁？"
他的语气冷得可怕。
她愣了一下："同事而已，怎么了？"
"同事？"他冷笑，"同事需要靠那么近说话？"
她这才反应过来："你吃醋了？"
他僵了一瞬，然后......脸红了。
```

## 六、言情写作禁忌

### 禁止项

| 禁止 | 原因 |
|------|------|
| 女主圣母 | 缺乏底线，读者厌恶 |
| 男主渣男 | 伤害女主后轻易原谅 |
| 误会拖太长 | 读者失去耐心 |
| 三角恋过多 | 喧宾夺主 |
| 强行虐 | 为虐而虐 |

### 警告项

| 警告 | 问题 |
|------|------|
| 女主太弱 | 依赖感过重 |
| 男主太完美 | 缺乏人味 |
| 互动单一 | 缺乏新鲜感 |
| 节奏太快 | 缺乏铺垫 |
