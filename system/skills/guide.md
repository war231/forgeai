# Workflow: Guide（灵活启动/存量整合）

> **来源**：武行 guide.md（增强版）

## 🧭 模式选择

询问用户当前创作状态：
1. **带资进组（存量整合）**：根目录下已有文件
2. **灵感切入（非线性启动）**：没有文件，想从特定角度开始
3. **常规流程**：返回标准模式

---

## 📂 模式 A: 带资进组

### Step 1: 资产扫描
列出项目根目录下所有文本文件（.md, .txt, .doc）

### Step 2: 智能识别与映射

| 文件内容特征 | 归档位置 |
|:---|:---|
| 样板书、同人文参考 | `1-边界/` |
| 故事梗概、核心脑洞 | `2-设定/2.1_创意脑暴.md` |
| 世界观、地图、等级体系 | `2-设定/2.2_新书设定案.md` |
| 角色介绍、人物小传 | `2-设定/2.4_主要角色设定表.md` |
| 金手指、系统说明 | `2-设定/2.3_金手指设定.md` |
| 分卷规划、细纲 | `3-大纲/` |
| 正文稿件 | `4-正文/` |

### Step 3: 执行迁移
移动/重命名或建立索引

### Step 4: 进度定锚
根据整合后的资产推荐下一步

---

## 🎨 模式 B: 灵感切入

| 场景 | 动作 |
|------|------|
| "只想先写人设" | 调用 ideation 步骤4 → 引导回流派 |
| "有绝妙核心梗/金手指" | 调用 ideation 步骤1/3 → 引导回流派 |
| "想直接写开头" | 警告+创建试读样章 → 引导补大纲/人设 |
