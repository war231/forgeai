# Workflow: Book Analyzer（样板书拆解）

> **功能**：分析样板书，提取结构、套路、节奏、文风
> **适用场景**：参考模式创作、对标学习、套路研究

## 📚 步骤 1: 输入准备

**交互式收集**：

1. **样板书来源**（三选一）：
   - A：粘贴文本（直接输入章节内容）
   - B：指定项目内路径（`1-边界/样板书/` 目录下）
   - C：指定样板书库路径（`../样板书库/{小说名}/` 目录下）

2. **分析范围**：
   - 章节范围：第 X 章到第 Y 章（建议 20-50 章）
   - 分析深度：快速（结构+爽点）/ 深度（全维度）

3. **分析重点**（可多选）：
   - [ ] 结构节奏
   - [ ] 爽点分布
   - [ ] 套路模式
   - [ ] 文风特征
   - [ ] 人物塑造
   - [ ] 伏笔技巧

**输出**：确认分析参数

---

## 🔍 步骤 2: 结构分析

**自动化分析**（调用 `book_analyzer.py`）：

### 2.1 章节结构

```
每章字数分布
├── 平均字数
├── 字数波动范围
└── 异常章节标记

章节节奏
├── 铺垫比例（约 30%）
├── 冲突比例（约 50%）
└── 高潮比例（约 20%）
```

### 2.2 卷结构

```
卷划分
├── 卷内章节数
├── 卷高潮位置
└── 卷收束方式
```

**输出**：`1-边界/样板书分析/结构分析.md`

## ⚡ 步骤 3: 爽点分析

**爽点识别与分类**：

### 3.1 爽点类型统计

| 爽点类型 | 出现次数 | 占比 | 典型章节 |
|---------|---------|------|---------|
| 装逼打脸 | N | XX% | 第X、Y、Z章 |
| 扮猪吃虎 | N | XX% | 第X、Y、Z章 |
| 越级反杀 | N | XX% | 第X、Y、Z章 |
| 身份掉马 | N | XX% | 第X、Y、Z章 |
| ... | ... | ... | ... |

### 3.2 爽点密度分析

```
平均密度：X 个/章
密度分布：
├── 高密度章节（3+个爽点）：第X、Y、Z章
├── 中密度章节（1-2个）：...
└── 低密度章节（0个）：第A、B、C章（检查是否拖沓）
```

### 3.3 爽点节奏曲线

```
章节 → 爽点数量 → 强度
1    → 2       → ★★☆
2    → 1       → ★☆☆
3    → 3       → ★★★
...
```

**输出**：`1-边界/样板书分析/爽点分析.md`
