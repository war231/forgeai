# PROJECT.md — ForgeAI

> **项目愿景**: 打造一个基于多Agent协作的长篇小说AI创作系统,通过RAG检索、智能审查、自动化流水线等技术,实现高质量的小说创作辅助。

---

## 项目概述

### 核心价值

ForgeAI是一个创新的长篇小说创作系统,通过以下技术实现高质量AI辅助创作:

1. **多Agent协作审查** - 9个专业Agent并行审查,确保质量
2. **RAG语义检索** - 智能召回伏笔、角色状态、相关章节
3. **自动化流水线** - 写前检查、上下文提取、写后处理全自动
4. **样板书学习** - 分析对标作品,提取套路/节奏/文风
5. **去AI痕迹** - 24种AI写作特征检测,保持文风自然

### 技术架构

```
VSCode Extension (TypeScript)
    ↓
Skill Layer (Markdown工作流)
    ↓
Agent Layer (9个专业Agent)
    ↓
Python Backend (29个核心模块)
    ↓
Data Layer (ChromaDB + SQLite + JSON)
```
